# kreja.github.io
This is my blog → [http://kreja.github.io/](http://kreja.github.io/)

### install

1. fork it 
2. rename your repository as `username.github.io`
3. clone your repository，and modify theme setting in `_config.yml`
4. just write your own posts!
5. It's done!
